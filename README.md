# Heart-Disease-Analysis-and-Prediction
# ❤️ Heart Disease Analysis Dashboard

An AI-powered Healthcare Analytics Dashboard built using Flask, Machine Learning, and Gemini AI.

This project analyzes heart disease data, provides interactive visualizations, predicts risk using a trained ML model, and generates AI-powered health reports.

---

## 🚀 Features

- 📊 Interactive Dashboard with Filters
- 📈 Data Visualization (Age, Gender, Lifestyle Factors)
- 🧠 Machine Learning Risk Predictor
- 🤖 AI-Generated Health Report (Gemini Integration)
- 📑 Data Summary API
- 🎯 Responsive UI Design

---

## 🛠️ Tech Stack

- Python
- Flask
- Pandas & NumPy
- Scikit-Learn
- Chart.js
- HTML, CSS, JavaScript
- Google Gemini API

## ⚙️ Installation

1. Clone the repository

2. Install dependencies

3. Run the app
python app.py

4. Open in browser
http://127.0.0.1:5000
